# Gala Cookbook

A collection of task-oriented recipes for solving problems with Gala. Each recipe is a self-contained `.gala` program with an accompanying walkthrough that explains the concepts, the code, and how to run it.

## Sections

| Section | Level | Topics |
|---------|-------|--------|
| [beginner/](./beginner/) | Getting started | Hello world, functions, control flow, imports |
| [quantum/](./quantum/) | Core quantum | Bell pairs, gates, superposition, entanglement, measurement |
| [hybrid/](./hybrid/) | Quantum-classical ML | Parameterized circuits, variational classifiers, gradients |
| [advanced/](./advanced/) | Deep dives | QFT, Grover's search, error handling, custom gates |
| [references/](./references/) | Concept guides | Effect system, linear types, compiler diagnostics |

See [RUNNING.md](./RUNNING.md) for instructions on compiling and running recipes.
