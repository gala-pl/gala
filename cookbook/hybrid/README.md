# Hybrid Recipes

Quantum-classical machine learning with Gala. These recipes use parameterized circuits, gradients, and the ML standard library.

| Recipe | What you'll learn |
|--------|-------------------|
| parameterized_circuits | Rotation gates with trainable parameters (`Params`) |
| variational_classifier | End-to-end variational classifier — encode, ansatz, measure |
| gradient_computation | Using `grad` to compute parameter-shift gradients |
| hybrid_classical_quantum | Classical pre/post-processing around quantum kernels |

**Prerequisites:** Quantum recipes or familiarity with qubits and gates.

**Key docs:** [TYPE_SYSTEM.md](../docs/TYPE_SYSTEM.md), [STANDARD_LIBRARY.md](../docs/STANDARD_LIBRARY.md)
