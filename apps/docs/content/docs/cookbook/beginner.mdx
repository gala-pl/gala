Classical programming fundamentals.

| Recipe | What you'll learn |
|--------|-------------------|
| hello_world | Your first Gala program — `fn main()`, `print()`, return values |
| classical_functions | Defining and calling functions, parameters, return types |
| control_flow | `if`/`else`, `match`, `for`, `while` — all expressions |
| modules_and_imports | Organizing code with modules and the `import` keyword |

**Prerequisites:** None. These recipes use only classical `pure` Gala.

All beginner recipes are in [`cookbook/beginner/`](https://github.com/anomalyco/gala/tree/main/cookbook/beginner).
