Qubits, gates, and the effect system.

| Recipe | What you'll learn |
|--------|-------------------|
| bell_pair | Create and measure a Bell state — entanglement in 5 lines |
| quantum_gates | Single-qubit gates (H, X, Y, Z, S, T) and two-qubit gates (CX, CZ, SWAP) |
| superposition | Put a qubit into superposition and understand probabilities |
| entanglement | Multi-qubit entanglement patterns and GHZ states |
| measurement_and_effects | Crossing the measurement boundary — `quantum` → `prob` effects |
| qubit_ownership | Linear types in action — no-cloning, move semantics, and `drop` |

**Prerequisites:** Beginner recipes or familiarity with Gala syntax.
