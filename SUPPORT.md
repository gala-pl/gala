# Support

## Community

- **Discussions**: GitHub Discussions
- **Discord**: [Join our server](https://discord.gg/gala-lang)
- **Stack Overflow**: Tag questions with `gala-lang`

## Bug Reports

Open an issue on GitHub with:
- A clear, descriptive title
- Steps to reproduce
- Expected vs actual behavior
- Gala version (`gala --version`)
- Rust version (`rustc --version`)

## Feature Requests

We track feature requests via GitHub Issues. Please search existing issues before opening a new one.

## Security

Report security vulnerabilities to **security@gala-lang.dev** — do not use public issues.
