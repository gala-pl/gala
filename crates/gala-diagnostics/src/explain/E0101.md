# E0101: Unexpected Token

The parser encountered a token it did not expect at this position in the source code.

## Example

```gala
fn main() -> Int {
    let x = 42 +
    return x;
}
```

## Explanation

The `+` operator requires an expression on its right-hand side, but the parser found `return` instead.

## Fix

Complete the expression:

```gala
fn main() -> Int {
    let x = 42 + 1;
    return x;
}
```
