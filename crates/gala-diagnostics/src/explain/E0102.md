# E0102: Unterminated String Literal

A string literal was started with `"` but never closed with a matching `"`.

## Example

```gala
fn main() -> Int {
    let msg = "Hello, world!
    return 0;
}
```

## Explanation

String literals must be on a single line and terminated with a closing quote. Use escape sequences for newlines inside strings.

## Fix

Close the string literal:

```gala
fn main() -> Int {
    let msg = "Hello, world!\n";
    return 0;
}
```
