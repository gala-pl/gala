# E0201: Type Mismatch

An expression was used where a different type was expected.

## Example

```gala
fn add(a: Int, b: Int) -> Int {
    return a + b;
}

fn main() -> Int {
    let x: Float = 3.14;
    let y = add(x, 2);  // error: expected Int, found Float
    return y;
}
```

## Explanation

The function `add` expects two `Int` arguments, but `x` is a `Float`. Gala does not perform implicit narrowing conversions.

## Fix

Either change the type of `x` to `Int`, or explicitly convert:

```gala
fn main() -> Int {
    let x: Int = 3;
    let y = add(x, 2);
    return y;
}
```
