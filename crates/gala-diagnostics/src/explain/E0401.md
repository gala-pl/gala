# E0401: Use After Move

A linear value was used after it was moved (consumed).

## Example

```gala
fn main() -> Qubit quantum {
    let q = qubit();
    let _ = measure(q);  // q is consumed by measure
    return q;            // error: use after move
}
```

## Explanation

Qubits are linear resources — they must be used exactly once. The `measure` function consumes the qubit (it's a destructive measurement). After measurement, the qubit no longer exists.

## Fix

Don't use the qubit after measuring it:

```gala
fn main() -> Measured<Bool> prob {
    let q = qubit();
    measure(q)  // q is consumed here, return the result
}
```
