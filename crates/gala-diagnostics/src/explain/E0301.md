# E0301: Effect Mismatch

A function with a higher effect was used where a lower effect was expected.

## Example

```gala
fn quantum_op() -> Qubit quantum {
    qubit()
}

fn pure_fn() -> Int pure {
    let q = quantum_op();  // error: quantum effect in pure context
    return 0;
}
```

## Explanation

Effects form a lattice: `pure ⊑ quantum ⊑ prob`. A function can only call functions with effects `⊑` its own. `pure` functions cannot call `quantum` or `prob` functions.

## Fix

Either change the effect of the calling function:

```gala
fn pure_fn() -> Int quantum {
    let q = quantum_op();
    return 0;
}
```

Or restructure to separate pure and quantum code.
